---

## 🖥️ Network Overview

This exercise demonstrates:

- Connecting end devices (PCs) to a switch  
- Connecting the switch to a router  
- Assigning IP addresses to each device  
- Verifying connectivity using `ping`  
- Understanding basic network components and how they interact
